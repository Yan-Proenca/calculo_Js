### Curso Técnico de Desenvolvimento de Sistemas - Senai Itapeva
# Calculadora de Exercícios
**Descrição:**
Aplicativo web com três exercícios de cálculo financeiro e de transporte, desenvolvido como projeto de prática para o curso técnico de Desenvolvimento de Sistemas do SENAI Itapeva. O projeto resolve problemas comuns de cálculo de horas extras, aumento salarial e custo de transporte, fornecendo uma interface simples e interativa.

## Funcionalidades
- Cálculo de valor total de horas extras (dias úteis e finais de semana).
- Cálculo de aumento salarial com faixas progressivas de percentual.
- Cálculo do custo mensal de transporte com descontos por quantidade de pessoas.
- Exibição de data e hora em tempo real nas páginas.
- Navegação entre os exercícios via menu principal.

## Tecnologias Utilizadas
- **Linguagem/Frameworks:**
 - HTML
 - CSS (TailwindCSS via CDN)
 - JavaScript

## Estrutura do Projeto
- `index.html` - Menu principal com links para os exercícios.
- `exercicio 1/horaExtra.html` + `exercicio 1/codigo.js` - Cálculo de horas extras.
- `exercicio 2/aumentoSalarial.html` + `exercicio 2/codigo.js` - Aumento salarial.
- `exercicio 3/custoTransporte.html` + `exercicio 3/codigo.js` - Custo de transporte.
